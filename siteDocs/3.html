When coding your website, there is a couple of options for what you code it in. The best program for beginners is notepad, it should already by downloaded if your Windows version is greater than 1.0. To open Notepad, press the Windows Key + R, type Notepad and press enter.

After opening Notepad, you will have a blank page. Your first line of code will be <!DOCTYPE html> this line tells the visitors of your page that it is being rendered in HTML. It isn't shown on your webpage, but is proper formatting.
